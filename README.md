## Installation

1. Clone this repository

    ```sh
    git clone https://github.com/jfang324/terminal-website.git
    ```

2. Install NPM packages

    ```sh
    npm install
    ```

3. Build and run with next.js if you want to check it out

    ```sh
    npm run build
    npm run start
    ```
